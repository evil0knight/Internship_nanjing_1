## 2/2

项目基本结束了

433模块加上FT61FC4芯片两个都休眠，平均电流45UA，比不休眠（21mA）减少467倍

433模块在物理层接受的协议中包含发送方地址，接受方通过解包数据链路层的帧提取地址判断是否接受信息

433模块用AT指令配置周期休眠，和前导码（可能是01010101。。。为了唤醒433而设置的长长的BIT串，）要设置长度根据数据手册和睡眠长度设置，要大一点，

注意USB转TTL供电较小，设置433功率的时候要考虑到电流大小，否则发送方会卡死

8位芯片的外部中断注意全局中断关断GIE=0，PEIE=0,EPIF0(外部管脚中断标志位)设置，都要根据例程和数据手册设置
